# Integração Completa - FastAPI + LangChain + LangGraph

## Exercício 1: API com LangChain

Integre LangChain em uma API FastAPI para processar perguntas.

## Exercício 2: API com LangGraph

Crie endpoints FastAPI que usam grafos LangGraph.

## Exercício 3: Streaming em API

Implemente streaming de respostas do LLM através da API.

## Exercício 4: WebSockets

Use WebSockets para comunicação em tempo real com o LLM.

## Exercício 5: Sistema Completo

Construa um sistema completo:
- Frontend (HTML/JS)
- Backend (FastAPI)
- LLM (LangChain/LangGraph)
- Banco de dados (opcional)

## Desafio Final

Crie um assistente virtual completo com:
- Interface web moderna
- API RESTful
- Agente LangGraph inteligente
- Histórico de conversas
- Múltiplas funcionalidades
