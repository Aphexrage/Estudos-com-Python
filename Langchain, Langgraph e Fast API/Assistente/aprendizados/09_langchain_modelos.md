# LangChain - Trabalhando com Diferentes Modelos

## Modelos Disponíveis

LangChain suporta múltiplos provedores: OpenAI, Anthropic, Cohere, HuggingFace, etc.

## Exercício 1: Comparando Modelos

Compare respostas de diferentes modelos (gpt-3.5-turbo, gpt-4, gpt-4o-mini) para a mesma pergunta.

## Exercício 2: Configurando Temperature

Experimente diferentes valores de temperature (0.0, 0.7, 1.0) e observe como afeta as respostas.

## Exercício 3: Modelos Locais

Configure um modelo local usando HuggingFace ou Ollama.

## Exercício 4: Streaming

Implemente streaming de respostas para melhor UX.

## Exercício 5: Retry e Error Handling

Configure retry automático e tratamento de erros para requisições de API.

## Desafio

Crie um sistema que automaticamente escolhe o melhor modelo baseado no tipo de tarefa.
