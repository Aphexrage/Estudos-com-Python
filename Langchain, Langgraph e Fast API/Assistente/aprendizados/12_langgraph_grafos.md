# LangGraph - Construindo Grafos

## Exercício 1: Grafo Linear

Crie um pipeline linear: validação -> processamento -> formatação -> saída.

## Exercício 2: Grafo com Branching

Implemente um grafo que se divide em múltiplos caminhos baseado em condições.

## Exercício 3: Grafo com Merge

Crie um grafo onde múltiplos caminhos convergem em um node.

## Exercício 4: Grafo com Feedback

Implemente um grafo com loop de feedback para refinamento.

## Exercício 5: Grafo Hierárquico

Construa um grafo com sub-grafos aninhados.

## Desafio

Crie um sistema de revisão de código que:
- Analisa código
- Identifica problemas
- Sugere correções
- Revisa novamente se necessário
- Gera relatório final
