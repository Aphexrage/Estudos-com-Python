# LangChain - Técnicas Avançadas

## Exercício 1: Retrieval Augmented Generation (RAG)

Implemente RAG para responder perguntas baseadas em documentos.

## Exercício 2: Agents

Crie um agente que pode usar ferramentas (calculadora, busca web, etc.).

## Exercício 3: Memory Management

Implemente diferentes tipos de memória (buffer, summary, entity).

## Exercício 4: Callbacks

Use callbacks para monitorar e logar execução de chains.

## Exercício 5: Custom Chains

Crie chains customizadas com lógica de negócio específica.

## Desafio

Construa um assistente que combina RAG, agents e memória para criar um sistema completo.
